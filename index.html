The first step is to rename the "master" branch in your *local* Git repositories: git branch -m master main 4. git branch New-Branch(this will add a new branch, New-Branch can be replaced with whatever name). 5. git checkout New-Branch(this will switch to a new branch, in this case you will switch to ‘New-Branch’. 6. git commit -m ‘Initial commit’ 7. git remote add [alias] [url] 8. git reset --hard Head, go to previous commit snapshot. 9. merging git merge [name of branch] 10.
